## 什么是 LifeRPG？

**LifeRPG** 是一个游戏化个人管理系统，它将你的日常任务、学习、运动、创作等活动转化为游戏中的任务，通过完成任务获得**经验值**、**金币**和**属性点**，让自我提升变得更有趣、更有动力。

### ✨ 核心特色

- 🎮 **完整的RPG系统**：等级、经验、属性、成就、金币
- 📊 **四维属性成长**：智力、活力、管理、创造
- ⚡ **属性衰减机制**：模拟现实，需要持续努力维持
- 🏆 **丰富的成就系统**：150+成就等你解锁
- 📅 **习惯打卡**：养成良好习惯，自动签到奖励
- 🎁 **奖励商店**：用金币兑换实体奖励，激励自己
- 🎨 **炫酷3D特效**：5000+粒子互动背景

## 快速开始

### 第一步：注册账号

1. 打开 LifeRPG
2. 点击「注册」
3. 输入用户名和密码
4. （可选）设置头像和个人信息

💡 **提示**：每个账号的数据独立存储，可以创建多个账号。

### 第二步：创建第一个任务

1. 进入「主页」
2. 点击右上角的「+ 新建任务」
3. 填写任务信息：
   - **任务标题**：例如"跑步30分钟"
   - **任务类型**：选择「日常」「主线」或「支线」
   - **关联属性**：选择「活力」（运动任务）
   - **奖励设置**：经验值 20，金币 15
4. 点击「创建」

### 第三步：完成任务

1. 去完成你的任务（例如真的去跑步）
2. 回到应用，找到刚才创建的任务
3. 点击任务卡片右侧的「✓ 完成」按钮
4. 🎉 恭喜！你将获得经验值、金币和属性提升

### 第四步：查看成长

- 查看你的等级是否提升
- 查看金币和属性的变化
- 前往「成就」页面，看看解锁了哪些成就

## 核心概念

### 💫 经验与等级

**如何获得经验？**
- 完成任务
- 每日登录
- 每日签到
- 解锁成就

**升级有什么好处？**
- 获得金币奖励
- 解锁新的商店物品
- 解锁更多成就
- 获得称号（等级 5/10/25/50/100 等）

**经验加成系统：**
- 🎯 **等级加成**：等级越高，完成任务获得的经验越多（最高1.3倍）
- 🔥 **连击加成**：连续登录天数越多，经验加成越高（最高1.5倍）

### 🎯 四维属性系统

LifeRPG 有四种核心属性，代表你的全面发展：

| 属性 | 名称 | 适用场景 | 示例任务 |
|------|------|----------|----------|
| **INT** | 智力 | 学习、科研、阅读 | 阅读技术书籍、学习课程、解决问题 |
| **VIT** | 活力 | 运动、健康、休息 | 跑步、健身、瑜伽、早睡早起 |
| **MNG** | 管理 | 规划、执行、组织 | 制定计划、完成工作任务、整理笔记 |
| **CRE** | 创造 | 创作、创新、艺术 | 写作、绘画、做视频、设计 |

**完成任务时：**
- 每个任务可以关联 1-4 个属性
- 完成后，每个关联属性 +10 点
- 属性值影响成就解锁

### ⚡ 属性衰减机制（重要！）

**为什么属性会衰减？**

在现实生活中：
- 今天运动了，体能提升，但长期不运动，体能会下降
- 学习了新知识，但不复习会遗忘
- 创作能力需要持续练习，否则会生疏

LifeRPG 模拟了这种现实规律：

| 属性 | 半衰期 | 衰减速度 | 建议 |
|------|--------|----------|------|
| INT（智力） | 14天 | 慢 | 每周至少完成1-2个学习任务 |
| VIT（活力） | 7天 | **最快** | 建议隔天运动，保持体能 |
| MNG（管理） | 10天 | 中等 | 持续规划和执行任务 |
| CRE（创造） | 12天 | 中等 | 定期进行创作活动 |

**如何应对衰减？**
1. 前往「属性平衡」页面查看健康度
2. 重点关注健康度 < 40% 的属性（红色警告）
3. 完成相关任务来恢复属性
4. 养成习惯，持续完成平衡的任务

### 💰 金币系统

**金币分类：**

LifeRPG 有两种金币类型：

1. **分类金币**（智慧币/活力币/管理币/创意币）
   - 完成对应属性的任务获得
   - 用于购买特定类型的奖励

2. **通用币**
   - 每次完成任务额外获得30%
   - 每日签到获得
   - 可以购买任何奖励

**示例：**
- 完成「跑步30分钟」任务（VIT属性，奖励100币）
  - 获得 100 活力币
  - 额外获得 30 通用币
  - 总金币增加 130

### 🏆 成就系统

成就分为以下类别：

- **里程碑**：等级、首次完成任务等
- **任务成就**：完成 10/50/100/500 个任务
- **连击成就**：连续登录 3/7/30 天
- **属性大师**：单项属性达到 50/100/250/500
- **特殊成就**：金币富豪、四维平衡等
- **隐藏成就**：夜猫子、早起鸟、收藏家等

**成就奖励：**
- 经验值
- 金币
- 成就点数
- 专属称号

## 功能详解

### 📝 任务系统

#### 任务类型

1. **主线任务**（蓝色）
   - 重要的长期目标
   - 如：完成项目、学习课程、健身计划

2. **支线任务**（绿色）
   - 次要但有价值的任务
   - 如：整理房间、看电影、社交活动

3. **日常任务**（黄色）
   - 每天重复的小任务
   - 如：阅读30分钟、运动、写日记

#### 任务优先级

- 🔴 **紧急**：截止日期临近的重要任务
- 🟠 **高**：重要但不紧急
- 🟡 **中**：一般任务
- 🟢 **低**：可延后的任务

#### 任务进度

- 设置0-100%的进度
- 达到100%自动完成
- 支持子任务拆分

#### 父子任务

**使用场景：**将大任务拆分为多个小任务

**示例：**
```
父任务：完成毕业论文
├─ 子任务1：文献调研
├─ 子任务2：撰写第一章
├─ 子任务3：撰写第二章
└─ 子任务4：论文修改
```

**如何创建：**
1. 创建父任务
2. 点击父任务的「添加子任务」
3. 完成所有子任务后，父任务自动完成

### 📅 习惯打卡

**习惯类型：**

1. **布尔型**：完成/未完成
   - 示例：早起、阅读、冥想

2. **数量型**：记录数值
   - 示例：喝水 2000ml、俯卧撑 50 个

3. **时长型**：记录时间
   - 示例：运动 30 分钟、学习 2 小时

**重复模式：**
- 每天
- 每周（选择星期几）
- 自定义（灵活选择日期）

**习惯统计：**
- 总完成次数
- 当前连续天数
- 最长连续天数
- 完成率

**自动签到机制：**

完成所有今日习惯 → 自动触发每日签到 → 获得签到奖励

### ✅ 每日签到

**签到奖励：**
- 基础：经验值 + 通用币
- 连续签到加成：
  - 7天：额外 50 exp + 各类币 10
  - 14天：额外 80 exp + 各类币 15
  - 30天：额外 200 exp + 各类币 40（大奖！）

**签到方式：**
1. 手动签到：前往「习惯打卡」页面点击签到
2. 自动签到：完成所有今日习惯自动签到

### 🛒 奖励商店

**商店分类：**

1. **消耗品**
   - 一次性使用
   - 如：电影票、美食、休闲活动

2. **永久物品**
   - 购买后永久保留
   - 如：书籍、游戏、装备

3. **限时物品**
   - 设置过期时间
   - 如：健身卡会员、课程

**购买条件：**
- 等级要求
- 金币要求（可指定币种）
- 成就点数要求
- 限购次数

**购买后：**
- 物品添加到背包
- 可以添加笔记（购买心得、使用体验）
- 记录使用历史

### 🎒 我的背包

**功能：**
- 查看已购买的物品
- 使用消耗品
- 添加/编辑笔记
- 查看使用历史
- 统计消费记录

**物品笔记：**
- 记录购买心得
- 记录使用体验
- 添加提醒事项
- 支持编辑和删除

### 📊 数据统计

**经验详情页：**
- 当前等级和经验进度
- 累计获得经验
- 经验来源 TOP 5

**金币详情页：**

**属性详情页：**

**属性平衡页：**（新功能！）

## 进阶技巧

### 🎯 高效任务管理

#### 1. 合理设置任务奖励

**经验值设置建议：**
```
小任务（15-30分钟）： 5-15 exp
中型任务（1-2小时）： 20-40 exp
大型任务（半天以上）： 50-100 exp
重大里程碑：        100+ exp
```

**金币设置建议：**
```
小任务：  5-15 币
中型任务： 20-40 币
大型任务： 50-100 币
```

#### 2. 任务优先级原则

使用「艾森豪威尔矩阵」：
- **紧急+重要** → 立即执行（优先级：紧急）
- **不紧急+重要** → 计划执行（优先级：高）
- **紧急+不重要** → 委托或快速处理（优先级：中）
- **不紧急+不重要** → 删除或延后（优先级：低）

#### 3. 使用番茄钟专注

在主页可以启动番茄钟：
- 25分钟工作 + 5分钟休息
- 完成4个番茄钟后，长休息15分钟
- 记录专注时长，计入统计

### ⚖️ 保持属性平衡

#### 每周任务模板

建议每周完成以下类型任务：

| 属性 | 建议频率 | 示例任务 |
|------|----------|----------|
| INT | 2-3次/周 | 阅读、学习课程、解决问题 |
| VIT | 3-5次/周 | 跑步、健身、瑜伽、早睡 |
| MNG | 2-3次/周 | 周计划、复盘、整理笔记 |
| CRE | 1-2次/周 | 写作、绘画、视频制作 |

#### 属性健康度监控

每周检查一次「属性平衡」页面：
1. 查看健康度评分
2. 重点关注 < 40% 的属性
3. 安排相关任务补强

### 💎 成就猎人策略

#### 快速解锁成就的技巧

1. **关注链式成就**
   - 完成初级成就自动解锁中级成就的挑战
   - 例如：完成10个任务 → 解锁完成50个任务挑战

2. **平衡发展解锁组合成就**
   - 「四维平衡者」：所有属性 ≥ 50
   - 「全能战士」：所有属性 ≥ 100

3. **注意隐藏成就**
   - 夜猫子：凌晨0-5点完成任务
   - 早起鸟：早上5-7点完成任务
   - 收藏家：解锁20个成就

### 🎁 商店购物策略

#### 1. 设置有吸引力的奖励

**示例：**
```
[消耗品]
- 电影票 (50 通用币)
- 奶茶 (20 活力币)
- 游戏时间 1小时 (30 创意币)

[永久物品]
- 《深度学习》书籍 (100 智慧币)
- Switch游戏 (200 通用币 + 50成就点)
- 机械键盘 (500 通用币)

[限时物品]
- 健身房月卡 (150 活力币，30天有效)
```

#### 2. 消费策略

- 优先使用分类金币（更容易积累）
- 通用币留给重要奖励
- 成就点数谨慎消费（较难获得）

## 常见问题

### Q1: 如何快速升级？

**A:**
1. 完成更多任务（主要经验来源）
2. 保持每日登录（连击加成最高1.5倍）
3. 提升等级（等级加成最高1.3倍）
4. 解锁成就（额外经验奖励）

### Q2: 属性衰减太快怎么办？

**A:**
这是设计机制，目的是鼓励持续努力：
- VIT（活力）衰减最快，建议隔天运动
- INT（智力）衰减最慢，每周学习2-3次即可
- 查看「属性平衡」页面监控健康度
- 重点关注健康度 < 40% 的属性

### Q3: 金币不够用怎么办？

**A:**
1. 完成更多任务
2. 保持签到连击（最高每天200exp+160币）
3. 设置合理的奖励金额（不要太贵）
4. 优先完成多属性任务（获得更多分类币+通用币）

### Q4: 如何养成习惯？

**A:**
1. 从小习惯开始（如每天喝水）
2. 使用重复模式（每天/每周）
3. 利用自动签到机制激励
4. 查看统计数据了解进度

### Q5: 忘记完成任务怎么办？

**A:**
1. 任务不会自动过期（除非设置截止日期）
2. 可以随时回来完成
3. 但连击会中断，签到连击也会重置
4. 建议设置提醒

### Q6: 可以删除/重置数据吗？

**A:**
- 删除任务：点击任务详情 → 删除
- 删除习惯：长按习惯 → 删除
- 重置游戏：设置 → 重置游戏（慎用！）
- 切换账号：点击头像 → 切换用户

### Q7: 数据安全吗？

**A:**
- 所有数据存储在浏览器本地（LocalStorage）
- 不会上传到服务器
- 每个用户独立存储
- 建议定期备份数据（导出功能开发中）

### Q8: 如何多人使用？

**A:**
系统支持多用户：
1. 注册多个账号
2. 每个账号数据独立
3. 切换账号：点击头像 → 选择用户
4. 所有数据完全隔离
