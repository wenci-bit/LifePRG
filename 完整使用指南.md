- 导出数据（JSON 格式）
- 导入数据（从备份恢复）
- 重置游戏（需二次确认）

**游戏统计：**
- 当前等级、经验、金币

## 🎯 使用流程

### 第一次使用

1. **启动应用**
   - 看到炫酷的欢迎页
   - 阅读四大特性介绍
   - 点击"开始你的征程"

2. **进入仪表盘**
   - 默认进入 Dashboard 页面
   - 查看当前状态（等级 1，所有属性 0）

3. **创建第一个任务**
   - 右侧任务日志中点击"新建任务"
   - 或点击左侧导航第二个按钮（Target）进入任务页面
   - 填写任务信息并创建

4. **完成任务**
   - 点击任务卡片标记完成
   - 观看粒子动画
   - 获得经验值和金币
   - 属性自动提升

5. **查看成就**
   - 点击左侧导航第三个按钮（Trophy）
   - 查看已解锁的成就
   - 追踪未完成成就的进度

## 💡 进阶技巧

### 任务管理策略

**每日早晨（5分钟）**
```
1. 打开 Quests 页面
2. 创建今日的 3-5 个 Daily 任务
3. 设置 1 个 Side 任务
```

**完成任务后（即时）**
```
1. 点击任务卡片完成
2. 观察属性提升
3. 检查是否解锁新成就
```

**每周回顾（周日晚）**
```
1. 进入 Achievements 查看解锁进度
2. 进入 Settings 查看统计数据
3. 导出数据备份
4. 规划下周的 Main Quest
```

## 📊 数据持久化

### 自动保存
- 所有操作自动保存到 localStorage
- 刷新页面不会丢失数据

### 手动备份
1. 进入 Settings 页面
2. 点击"导出"按钮
3. 保存 JSON 文件到本地

### 恢复数据
1. 进入 Settings 页面
2. 点击"导入"按钮
3. 选择之前导出的 JSON 文件

## 🐛 故障排除

### Q: 页面切换没有反应？
**A:** 刷新浏览器，确保使用最新代码

### Q: 成就进度不更新？
**A:** 完成相关任务后会自动更新，检查任务类型是否正确

### Q: 导出的数据能在其他浏览器使用吗？
**A:** 可以！导出JSON后，在任何浏览器通过导入恢复

### Q: 如何快速升级？
**A:** 创建多个高经验值的 Main Quest（50-100 EXP）
