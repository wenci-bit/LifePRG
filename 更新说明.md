# LifeRPG 优化更新说明

**更新内容：**

- 更密集的粒子云

- 保持高性能（使用 InstancedMesh）

### 3. 白天/黑夜主题切换 ☀️🌙

**全新主题系统：**

#### 主题切换按钮
- 位置：右上角固定按钮
- 图标：月亮（黑夜）/ 太阳（白天）
- 动画：旋转切换效果
- 悬停：缩放 + 背景光效

#### 黑夜模式（默认）
- 背景：深空灰 `#050505`
- 文字：白色 `#ffffff`
- 玻璃卡片：半透明白色 `rgba(255,255,255,0.05)`
- 边框：透明白色

#### 白天模式
- 背景：明亮灰蓝 `#f0f4f8`
- 文字：深色 `#1a202c`
- 玻璃卡片：半透明白色 `rgba(255,255,255,0.7)`
- 边框：淡黑色
- 增强阴影效果

#### 技术实现
- 使用 Zustand 状态管理
- LocalStorage 持久化（刷新保持）
- CSS 变量动态切换
- 平滑过渡动画（0.3s）
- 全局主题应用

## 🐛 已知问题 & 解决方案

### Q: 粒子太多导致卡顿？
**A:** 降低粒子数量到 5000-8000
```typescript
<ParticleSystem count={6000} />
```

### Q: 白天模式某些文字看不清？
**A:** 调整 globals.css 中的颜色对比度

### Q: 每日名言想要固定一条？
**A:** 修改 HeroPage.tsx:131-136
```typescript
const dailyQuote = QUOTES[0]; // 固定第一条
```
