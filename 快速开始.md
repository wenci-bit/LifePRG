# LifeRPG 快速开始指南

## 📦 安装与运行

### Step 1: 安装依赖

```bash
cd sys5
npm install
```

> ⏱️ 预计耗时：2-3 分钟

### Step 2: 启动开发服务器

```bash
npm run dev
```

### Step 3: 打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

---

## 🎯 首次使用教程

### 1️⃣ 创建你的第一个任务

1. 点击右侧的 **"新建任务"** 按钮
2. 填写表单：
   ```
   任务标题: 阅读论文 2 篇
   任务描述: 重点关注方法论部分
   任务类型: Daily (日常)
   关联属性: INT (智力)
   经验值奖励: 20
   金币奖励: 10
   ```
3. 点击 **"创建任务"**

### 2️⃣ 完成任务获得奖励

- **点击任务卡片**，任务会被标记为完成
- 你将获得：
  - ✨ 20 EXP
  - 💰 10 Gold
  - 🧠 INT +10

### 3️⃣ 观察变化

- 左侧 **HUD** 显示经验值增加
- 中间 **雷达图** 显示 INT 属性提升
- 完成 5 个任务后即可升级！

## 💡 使用技巧

### 任务设计建议

| 任务类型 | 经验值范围 | 适用场景 |
|---------|-----------|----------|
| **Daily** | 10-20 | 每天可重复的习惯 |
| **Side** | 20-50 | 一次性的小目标 |
| **Main** | 50-100 | 重要的长期目标 |

### 属性平衡策略

```
理想状态：四个属性都在 50-80 之间
避免：单一属性过高（如 INT 200，其他都是 0）
建议：每周至少提升一次每个属性
```

### 金币兑换建议

- 积累 50 Gold：兑换一次 "Cheat Meal"
- 积累 100 Gold：兑换 "周末看电影"
- 积累 200 Gold：兑换 "购买一本新书"

## 🐛 常见问题

### Q: 粒子背景不显示？

**A:** 检查浏览器是否支持 WebGL

```bash
# 在浏览器控制台执行
console.log(!!window.WebGLRenderingContext);
# 返回 true 表示支持
```

### Q: 数据丢失了？

**A:** 数据存储在浏览器的 localStorage 中，清除缓存会导致数据丢失。

**解决方案（未来功能）：** 导出数据到 JSON 文件

### Q: 如何重置所有数据？

**A:** 在浏览器控制台执行：

```javascript
localStorage.removeItem('liferpg-storage');
location.reload();
```

### Q: 任务类型如何选择？

**A:**
- **Main Quest**: 需要数周/数月完成的大目标
- **Side Quest**: 需要数天完成的中等目标
- **Daily Quest**: 每天可以重复的习惯养成

## 📈 进阶玩法

### 1. 周计划模式

每周日晚上，规划下周的任务：

- 1-2 个 Main Quest（长期目标的里程碑）
- 3-5 个 Side Quest（本周要完成的事项）
- 7-10 个 Daily Quest（每日习惯）

### 2. 属性专项突破

选择一个月专注提升某一属性：

- **INT 月**：疯狂阅读论文、学习新技能
- **VIT 月**：每天运动、规律作息
- **MNG 月**：整理文档、优化工作流
- **CRE 月**：撰写博客、尝试新爱好

### 3. 金币挑战

设定目标：

- 本周赚取 100 Gold
- 本月赚取 500 Gold
- 存钱购买 "大奖励"（如：周末旅行 = 1000 Gold）
