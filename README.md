## 🎯 项目概述

LifeRPG 是一款将现实生活任务转化为 RPG 游戏机制的个人管理系统。通过**游戏化（Gamification）**和 **AI 技术**，将科研、健康、学习、社交等日常活动数值化、可视化，帮助你更有动力地管理自己的人生。

### 核心理念

- 🎮 **角色扮演化生活管理** - 你就是主角，生活就是游戏
- 📊 **四维属性系统** - INT/VIT/MNG/CRE 全方位成长
- 🤖 **AI 智能助手** - 个性化任务推荐和智能奖励设定
- ✨ **视觉化反馈** - 3D 粒子特效 + 动态雷达图
- 🏆 **成就驱动** - 任务分级、经验值、金币奖励

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0
- npm 或 yarn 或 pnpm
- 现代浏览器（Chrome/Edge/Firefox）

### 安装步骤

```bash
# 1. 进入项目目录
cd sys5

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

### 访问应用

打开浏览器访问 [http://localhost:3000](http://localhost:3000)

### 首次使用

1. **注册账号** - 创建你的游戏角色
2. **完成引导** - 设定身份、目标和任务强度（2分钟）
3. **使用 AI** - 点击"AI 建议"获取个性化任务
4. **开始成长** - 完成任务，提升属性，解锁成就

**[详细使用指南 →](./快速启动指南.md)**

## ✨ v2.0 新功能

### 1. 🎯 智能用户引导

## ✨ 核心特性

### 1. 🎨 3D 粒子点云背景

- **5000+ 粒子**的动态点云球体
- 青色 → 紫色渐变效果
- **鼠标交互**：移动鼠标产生力场排斥效果
- 性能优化：使用 `InstancedMesh` 技术

### 2. 📊 四维属性系统

| 属性 | 英文缩写 | 对应领域 | 提升方式 |
|------|----------|----------|----------|
| 智力/科研 | INT | 论文阅读、代码编写、实验进度 | 完成学术任务 |
| 活力/健康 | VIT | 睡眠、运动、饮食 | 健身、早睡、健康饮食 |
| 管理/规划 | MNG | 按时完成任务、整理文档 | 时间管理、任务规划 |
| 创造/灵感 | CRE | 撰写博客、头脑风暴、艺术爱好 | 创意活动 |

### 3. 🎯 任务系统

#### 任务分级
- **Main Quest (主线任务)** - 高经验值、长期目标
- **Side Quest (支线任务)** - 中等经验值、一次性任务
- **Daily Quest (日常任务)** - 低经验值、可重复任务

#### 任务完成流程
```
添加任务 → 完成任务 → 粒子爆炸动画 → 获得奖励 → 属性提升 → 经验增加 → 升级！
```

## 🎮 使用指南

### 1. 创建你的第一个任务

1. 点击 "新建任务" 按钮
2. 填写任务信息：
   - **任务标题**：例如 "完成论文第三章"
   - **任务描述**：详细说明
   - **任务类型**：Main / Side / Daily
   - **关联属性**：INT / VIT / MNG / CRE

3. 点击 "创建任务"
